## API #api 
Segun **red hat**
	Una api es un conjunto de reglas y protocolos utilizados para el diseño y creación de servicios web, permitiendo la comunicación entre diferentes aplicaciones de software a traves de internet
En nuestro caso usaremos el #api como Backend
## REST #rest
Este nuevo enfoque de desarrollo de proyectos y servicios web fue definido por Roy fielding, el padre de la especificación **HTTP** y uno de los referentes internacionales en todo lo relacionado con la arquitectura de redes
