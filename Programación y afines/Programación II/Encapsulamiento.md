---
- ## Encapsulamiento $\to$ ocultar
--------
Datos de acceso 
- *Public*
Datos de solo uno $[Privados/Admin]\to$ *Titular* $\to$ *Clase*
- *Private*
Datos de poco acceso 
- *Protected*
---
Esta definición puede aplicarse a:
- *Clases*
- *Variables*
- *Métodos*
## Sintaxis
- *Encapsulado tipo_de_dato nombre_Variable* 

## Métodos
- *Get* $\to$ **Obtener**
	- Métodos puente para acceder a los elementos privados.
- *Set* $\to$ **Asignar**
	- Métodos para asignarle un valor a una variable privada.
